# Giving Report

A Better Tithe.ly Report Generator

A small app that produces a better report than what Tithe.ly generates. Everything is generated in the browser. So none of you information is stored.

1. Make sure the required fields are selected.<br>Required Fields: Net Amount, Name, Fees, Status, Covered Fees, Fund Name, Payment Type, Check #
2. Download the CSV.
3. Upload the CSV to giving.report.
4. Print your report or save as a PDF.
